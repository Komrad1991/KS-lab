# Command Palette для TiniJS

## Описание задачи

**TiniJS** — это фреймворк для создания веб-приложений на основе Web Components.  
Необходимо реализовать систему **Command Palette (командной панели)**, которая позволяет:

- Быстро находить команды через поиск
- Выполнять команды
- Использовать клавиатурные сокращения

---

## Требования

### 1. Базовая система Command Palette

Создать компонент `CommandPalette` в пакете `@tinijs/ui`.

**Функциональность:**

- Открытие по `Cmd/Ctrl + K`
- Оверлей с модальным окном
- Поле поиска с поддержкой fuzzy search
- Список команд:
  - иконка
  - название
  - описание
- Навигация:
  - `↑ / ↓` — перемещение
  - `Enter` — выполнение
- Закрытие:
  - `Esc`
  - клик вне окна

---

### 2. Система регистрации команд

Создать сервис `CommandService` в пакете `@tinijs/toolbox`.

**API:**

```ts
registerCommand(
  id: string,
  name: string,
  description: string,
  icon: string,
  action: () => void | Promise<void>,
  keywords?: string[]
)
```

**Возможности:**

- Группировка команд по категориям
- Динамическая регистрация / удаление
- Условное отображение (в зависимости от контекста/страницы)

---

### 3. Система клавиатурных сокращений

Расширить `CommandService`.

**API:**

```ts
registerShortcut(
  keys: string,
  commandId: string,
  preventDefault?: boolean
)
```

**Поддержка:**

- Модификаторы:
  - Ctrl
  - Alt
  - Shift
  - Meta
- Комбинации клавиш
- Последовательности:
  - пример: `"g h"` → "go home"
- Обработка конфликтов
- Отображение сочетаний в UI

---

### 4. Интеграция с роутером

Интеграция с `@tinijs/router`.

**Функциональность:**

- Автоматическая регистрация команд для роутов
- Формат:
  ```
  Перейти к [Название страницы]
  ```
- Быстрый доступ к недавно посещенным страницам

---

### 5. UI компоненты

Создать:

- `tini-command-palette` — модальное окно
- `tini-command-item` — элемент списка
- `tini-shortcut` — отображение горячих клавиш

**Стилизация:**

Поддержка 9 тем:

- Bootstrap
- Material
- iOS
- Fluent
- Radix
- Chakra
- Daisy
- Shadcn

Дополнительно:

- Светлая / тёмная тема

## Ожидаемый результат

Пользователь может:

- Открыть Command Palette (`Cmd/Ctrl + K`)
- Найти команду через поиск
- Перемещаться по списку (`↑ ↓`)
- Выполнить команду (`Enter`)
- Использовать горячие клавиши
- Видеть историю команд
- Быстро переходить по страницам приложения
