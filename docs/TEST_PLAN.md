# Тест-план для Command Palette и Command Service

1. Unit-тесты
- CommandService: регидистрации, unregister, группировки, shouldShow, getRegisteredCommands
- Hotkeys: registerShortcut, конфликт-контроль, preventDefault
- Базовые сценарии навигации и вызова команд

2. Тесты fuzzy search
- Проверка релевантности; тесты на частоту использования; граничные случаи (нет совпадений)
- Тесты по transliterate/алгоритму поиска

3. Тесты клавиатурных событий
- Обработка клавиш ArrowUp/ArrowDown, Enter, Esc, сочетания модификаторов
- Поддержка последовательностей (например, "g h")

4. Интеграционные тесты
- Регистрация автоматических команд роутера и навигационных
- Отображение палитры и выбор по Enter

5. Примеры использования
- В apps/ добавление простого примера использования палитры

Цели покрытия: минимальные 80% по критичным путям, тестовая среда настроена на CI.
