# Test Plan: Command Palette и Command Service

1. Unit-тесты
- CommandService: register/unregister, getRegisteredCommands, группировки, shouldShow, инициализация контекста.
- Hotkeys: registerShortcut, конфликт-контроль, optional preventDefault.
- Базовые сценарии вызова команд и их регистрации.

2. Тесты fuzzy search
- Проверка релевантности и частоты использования; граничные случаи (нет совпадений).
- Тесты transliterate/алгоритма поиска.

3. Тесты клавиатурных событий
- Обработка ArrowUp/ArrowDown, Enter, Esc, модификаторов и последовательностей (например, "g h").
- Проверка конфликта и отмены по keydown.

4. Интеграционные тесты
- Автоматическая регистрация навигационных команд роутеров и их отображение в палитре.
- Открытие палитры, навигация и выполнение через Enter.
- Проверка сохранения истории команд в localStorage.

5. Примеры использования
- В apps/ добавляем простой пример использования палитры.

6. Покрытие и CI
- Цель покрытия: минимальные 80% по критичным путям. Настройка CI соответствующая проекту.
