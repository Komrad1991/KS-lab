# UI Компоненты Command Palette

1. tini-command-palette
- Основной модальный компонент палитры.
- Открытие по Cmd/Ctrl+K; закрытие Esc и кликом вне окна.
- Поле поиска с fuzzy-search; навигация стрелками; Enter — выполнение.
- Отображение истории и недавно посещённых страниц.
- Визуальная подсветка текущего выбора; рядом отображаются клавиатурные сокращения.
- Поддержка светлой/темной темы; стилизация под 9 тем оформления.

2. tini-command-item
- Элемент списка команды: отображает иконку, название, описание, shortcut.
- Поддержка индикатора загрузки (spinner) для асинхронных команд.
- Клик по элементу — выполнение команды.

3. tini-shortcut
- Компонент отображения клавиатурного сокращения: визуализация последовательности (например, "g h").
- Поддерживает несколько вариантов отображения в зависимости от темной/светлой темы.

4. API совместимость
- Компоненты соответствуют интерфейсам из @tinijs/ui.
- Хуки доступа к глобальному состоянию истории через createStore.

5. Стили и темизация
- Поддержка 9 тем оформления и адаптивности под desktop/mobile.
- Поддержка светлой и темной тем.

Примечание: документация по API будет развита после утверждения архитектуры.
