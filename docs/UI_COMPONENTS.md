# UI Компоненты для Command Palette

1. tini-command-palette
- Основной модальный компонент палитры.
- Поддержка открытия по Cmd/Ctrl+K, закрытия по Esc и клику вне окна.
- Поле поиска с fuzzy-search; список команд с навигацией стрелками; Enter — выполнение.
- Отображение истории и недавно посещённых страниц.
- Визуальная подсветка текущего выбора; отображение shortcuts рядом с командами.
- Поддержка тем: светлая/темная.

2. tini-command-item
- Элемент списка команды: отображает иконку, название, описание, shortcut.
- Поддержка загрузки индикатора для асинхронных команд (spinner).

3. tini-shortcut
- Компонент отображения клавиатурного сокращения: визуализация последовательности (например, "Ctrl K" или "g h").

4. API-совместимость
- Компоненты должны соответствовать базовым интерфейсам из @tinijs/ui.
- Хуки для доступа к глобальному состоянию истории через createStore.

5. Стили и темизация
- Поддержка 9 тем оформления и адаптивности под desktop/mobile.

Примечание: документация по API будет развита после утверждения архитектуры.
